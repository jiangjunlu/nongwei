水稻

　　1.东农421：东北农业大学农学院选育。生育日数137d，需活动积温2550℃～2650℃。株高90cm～95cm，穗长16.5cm，每穗粒数80粒～90粒，千粒重26.8g，主茎叶数13片。分蘖力强，抗倒伏性中等，耐寒性较强。适应区域为黑龙江省第一积温带和第二积温带上限插秧栽培。 2．东农422： 东北农业大学农学院选育。生育日数132d，需活动积温2592℃。株高86.5cm，穗长19.5cm，每穗粒数90粒～100粒，千粒重26g～28g，主茎叶数12片。耐寒性强，秆强，抗倒伏。该品种叶色淡绿，故不要误认为缺肥追加过量氮肥，以防稻瘟病。适应区域为黑龙江省第二积温带下限和第三积温带上限插秧栽培。 3.东农423号：东北农业大学农学院选育。生育日数138 d～140d，需活动积温2600℃～2650℃。株高86 cm～95cm， 穗长21cm～26cm，每穗粒数110粒～120粒，千粒重26g～28g。粒长粒大，无芒，秆粗、活秆成熟。适应区域为黑龙江省第一积温带插秧栽培。 4.龙粳10号： 黑龙江省农业科学院水稻研究所选育。生育日数130d左右，需活动积温2400℃。株高90cm～95cm，穗长16cm～17cm，每穗粒数80粒左右，千粒重24g，主茎叶数11片～12片。分蘖力中等，后熟快，结实率高。适应区域为黑龙江省第二积温带插秧栽培。 5.龙粳11号：黑龙江省农业科学院水稻研究所选育。生育日数125d～130d左右，需活动积温2350℃。株高85cm～90cm，每穗粒数90粒，千粒重27g，主茎叶数11片。前期耐低温，出苗快，分蘖力强，茎秆粗壮，抗倒伏。适应区域为黑龙江省第二积温带下限和第三积温带上限插秧栽培。
